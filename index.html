Guia da Atividade:

Introdução ao HTML

Criação de Titulos

Temos as Tags que vão de <h1> a <h6> em HTML que definem os Títulos.

Experimenta criar Títulos escrevendo assim:

<h1>Titulo 1</h1>

Altera o <h1> e o </h1> para h2, h3, h4, h5, h6

Criação de Parágrafos

Para criares Parágrafos basta usares a Tag <p> finalizando sempre com a Tag de fecho </p>

Experimenta criar Parágrafos escrevendo assim:

<p>Parágrafo 1</p>

Introdução ao HTML

Criação de Listas

Para criares Listas tens de usar duas Tags <ul> e <li> finalizando sempre com a Tag de fecho. Como temos duas Tags primeiro fecha-se a última Tag que foi aberta e depois a primeira, </li> e </ul>

Experimenta criar uma Lista escrevendo assim:

<p>Lista de Compras:</p>
<ul>
<li>Maçã</li>
<li>Banana</li>
</ul>

Introdução ao CSS

Vamos usar o código que acabámos aprender mas com estilo!!

Para usarmos o CSS temos uma Tag especial chamada <style>, ao usarmos esta Tag podemos mudar tudo o que quisermos, desde cor do texto, fonte do texto, entre muitas outras.

Para começares a usar o CSS escreve assim:

<h1 style="color:red" >Titulo 1</h1>

<p style="color:blue" >Parágrafo 1</p>

<p>Lista de Compras:</p>
<ul>
<li style="color:green" >Maçã</li>
<li style="color:yellow" >Banana</li>
</ul>

Adicionar Imagem e Descrição

Para adicionares uma imagem tens de usar a Tag <img>.

Para começares a criar Imagens escreve assim:

<img src="URL da imagem escolhida" alt="Insere aqui o nome do teu TEMA">

Agora adiciona uma legenda à tua imagem usando o que já aprendeste.

  • Cria um parágrafo a seguir à imagem.

    • Já criaste a tua imagem com uma legenda?? Agora vamos pô-la dentro de uma caixa!!

    Para criares a caixa e inserires a imagem basta usar uma nova Tag <div>. Não te esqueças esta Tag precisa de ser fechada.

    <div class="gallery">
    <img src="URL da imagem escolhida" alt="Insere aqui o nome do teu TEMA">
    <p style="color:blue" >Parágrafo 1</p>
    </div>

    Edição de Imagem

    Agora vamos editar a imagem ao nosso gosto.
    Para isso edita o style, experimenta alterar a borda da imagem (border-radius) e altera o tamanho da imagem, (width).
    Não te esqueças também de alterar a cor do parágrafo.

    Dica: Para alterares o tamanho da fonte do parágrafo podes adicionar à frente da cor o seguinte style font-size:24px

    <img src="URL da imagem escolhida" alt="Insere aqui o nome do teu TEMA"
    style="border-radius:50%; width:20%">
    <p style="color:red" >Parágrafo 1</p>

    Adicionar um Video do YouTube

    Para adicionar um video do YouTube usamos uma Tag <iframe>.
    Esta Tag serve para criar uma moldura do video.

    Sendo assim vamos criar uma frame sem bordas e que permita fazer ecrâ inteiro com o video que escolheste.

    Para usar o URL video YouTube tens de primeiro esolher o video, de seguida copias só o código do video que se encontra a seguir ao =.

    Exemplo:

    https://www.youtube.com/watch?v=YSoT3T58QFY

    <iframe id="video" width="420" height="345" src="https://www.youtube.com/embed/CÓDIGO do video escolhido" style="border:none" allowfullscreen>

    Adicionar Fundo com Cores e Imagens


    Para adicionares um fundo colorido vamos criar uma caixa e preenche-la com cores:

    <div style="height:30vh;background-color:red" ></div>

    <div style="height:70vh;background-image:url('URL da tua Imagem')" ></div>

    Inversão de uma Imagem em relação aos eixos dos XX e dos YY


    Primeiro antes de inverter-mos uma imagem precisamos de criá-la junto com a chamada de funções para mais à frente podermos criar funções que nos irão alterar os eixos da imagem

    <img id="imagem" alt="imagem" src="URL da tua Imagem" >
    <br>
    <p><b>Inverter em relação às abcissas.</b></p>
    <a id="abcissas" onmouseover="mouseOverX()" onmouseout="mouseOutX()" >Inverter X</a>
    <br>
    <p><b>Inverter em relação às ordenadas.</b></p>
    <a id="ordenadas" onmouseover="mouseOverY()" onmouseout="mouseOutY()" >Inverter Y</a>

    No exercício a seguir vamos usar uma nova Tag chamada de <script>.
    Nesta Tag podemos criar funções e nela podemos fazer tudo o que quisermos

    Introdução ao JavaScript

    Continuação do Exercício anterior com animações

    Para este exercício vamos inverter uma imagem em relação aos eixos das abcissas e das ordenadas.
    Sendo assim basta transformar a imagem em relação a um dos eixos com uma escala de 1 e -1.

    <script>
    function mouseOverX() {
    document.getElementById("abcissas").style.color = "red"; document.getElementById("imagem").style.transform = "scaleX(-1)"; }

    function mouseOutX() {
    document.getElementById("abcissas").style.color = "black"; document.getElementById("imagem").style.transform = "scaleX(1)"; }
    </script>
